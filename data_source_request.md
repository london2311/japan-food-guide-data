# Roadmap

## v0.1.x - Stabilize the prototype

- Add schema validation for `restaurants.csv` and `restaurants.json`
- Add link validation for official and source URLs
- Add better error messages for malformed HTML
- Add Google Colab tutorial for beginners
- Add GitHub Pages sample viewer

## v0.2.x - Real-world public sources

- Add Hiroshima tourism association collector
- Add municipal food/tourism page collector template
- Add official restaurant website parser template
- Add source freshness checks
- Add polite request throttling and robots.txt notes

## v0.3.x - Travel-friendly exports

- Add multilingual export helper
- Add dietary option normalization
- Add accessibility fields
- Add map-friendly GeoJSON export
- Add sample datasets for Kyoto, Osaka, and Tokyo

## Maintenance workflow

The project will be maintained through GitHub Issues, pull requests, tests, releases, and scheduled GitHub Actions runs. Codex can support issue triage, PR review, collector implementation, test creation, documentation updates, and release preparation.
