# 法務・利用規約・マナーに関する注意

このプロジェクトは、訪日旅行者に役立つ公開飲食店情報を整理するためのOSSです。
利用時は、必ず対象サイトのルールを確認してください。

## 避けるべきこと

- 口コミサイトのレビュー本文を無断保存する
- 第三者の写真を無断保存・再配布する
- 独自評価点やランキングを無断複製する
- ログインが必要なページを自動収集する
- 短時間に大量アクセスする
- robots.txtや利用規約に反する収集をする

## 推奨する情報源

- 店舗公式サイト
- 自治体の観光ページ
- 観光協会の公式ページ
- 商店街や地域団体の公式ページ
- オープンデータとして提供されている情報

## アクセス頻度

GitHub Actionsなどで定期実行する場合も、頻度は控えめにしてください。
まずは1日1回以下を目安にし、対象サイトに負荷をかけないことを優先します。
